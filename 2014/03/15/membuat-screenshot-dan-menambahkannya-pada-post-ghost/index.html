Menambahkan screenshot image di Ghost editor

Saturday, March 15, 2014

Untuk membuat screenshot, saya biasanya menggunakan sebuah aplikasi gratis bernama Scitch. Nah, untuk menambahkan image screenshot di Ghost editor ternyata sangat mudah.

Caranya:

  1. Buat placeholder image di Ghost editor dengan cara: ![]
  2. Buat screenshot dengan Scitch.
  3. Posisikan window Browser dan Scitch bersebelahan sehingga mudah untuk melakukan drag-n-drop.
  4. Dari window Scitch, drag icon jpg pada bagian tengah bawah dan drop pada placeholder yang telah anda buat di Ghost editor
  5. Screenshot anda secara otomatis langsung ditambahkan dalam postingan.

Mudah sekali bukan? :)

Bekerja dengan Ghost

Setelah selesai menginstal Ghost di laptop saya, saya pikir perlu untuk membuat beberapa catatan. Biar nggak lupa, atau mungkin bisa berguna untuk kawan2 yang ingin mencoba bekerja dengan Ghost.

Berikut ini adalah langkah2 untuk bekerja dengan Ghost:

  1. Aktifkan HTTP server, dalam hal ini saya menggunakan XAMPP. Untuk bekerja dengan Ghost, anda cukup jalankan Apache servernya saja.
  2. Buka terminal window dan masuk ke directory Ghost, di laptop saya: /Applications/XAMPP/xamppfiles/htdocs/ghost
  3. Jalankan Ghost dengan cara: npm start di command line
  4. Login ke Ghost melalui browser: http://localhost:2368/ghost
  5. Buat postingan baru atau edit postingan lama. Jangan lupa meng-click tombol "Publish now"
  6. Pastikan anda berada pada directory Buster, contoh pada laptop saya: ~/buster/buster, lalu generate static pages dengan cara: buster generate --domain=http://127.0.0.1:2368.
  7. Upload ke Github dengan cara: buster deploy
  8. Tampilkan di browser dengan URL: http://asambul.github.io/ (lihat screenshot pada top image)

Catatan: Jika sebelumnya anda melakukan perubahan di remote repository sehingga isi direktori static di lokal berbeda dengan yang ada di remote (GitHub), masuklah ke direktori static dan jalankan perintah: git pull untuk men-sync repository lokal dan remote.

Sejauh ini, menurut saya Ghost cukup baik dan mudah digunakan. Kita hanya perlu membiasakan diri dulu dengan beberapa commands. Kalau sudah hafal, membuat postingan dapat dilakukan dengan cepat dan mudah.

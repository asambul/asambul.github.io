Kembali ke Github

Hari ini saya mengarahkan lagi DNS CNAME alwinsambul.info dari Bitnami ke repository Github saya. Penyebabnya adalah, tiba-tiba saja upload image di Ghost tidak bisa dilakukan. Pesan kesalahannya "Something went wrong!" sangat membingungkan dan kelihatannya tidak ada jalan lain kecuali menginstall lagi Ghost dari awal. Jadinya saya memutuskan untuk kembali ke Github.

Jadilah kondisi saat ini adalah http://www.alwinsambul.info mengarah ke http://asambul.github.io. Untuk mengedit blog Ghost saya harus melakukannya dari Localhost, men-generate static pages menggunakan Buster, lalu mem-push perubahan ke Github repository.

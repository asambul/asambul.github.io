Menjalankan Ghost dengan Forever

Apa itu Forever?

Forever adalah sebuah modul Node yang dapat digunakan untuk menjalankan Ghost secara permanen. Forever juga akan memonitor apakah Ghost berjalan atau tidak. Jika misalnya Ghost mengalami crash, Forever akan secara otomatis menjalankan Ghost kembali.

Instalasi Forever

Forever dapat di-instal dengan menjalankan command berikut:

 npm install -g forever

Jika gagal, kemungkinan karena anda butuh akses administrator untuk menjalankan perintah di atas. Coba lagi dengan menggunakan: sudo <command>

Menjalankan Ghost dengan Forever

Masuk dulu ke direktori Ghost, contoh: /Applications/XAMPP/htdocs/ghost pada Mac saya. Selanjutnya jalankan perintah berikut:

 forever start index.js

Anda dapat menutup command window, Ghost akan tetap berjalan di background.

Menghentikan Ghost dengan Forever

Pastikan anda berada pada direktori Ghost (lihat langkah sebelumnya), lalu jalankan perintah berikut:

 forever stop index.js

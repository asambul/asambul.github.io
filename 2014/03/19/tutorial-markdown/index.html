Tutorial Markdown (Bagian 1)

Pendahuluan

Markdown adalah modul editor di Ghost yang tugasnya menerjemahkan teks yang kita masukkan menjadi kode HTML. Dengan Markdown, berbagai fungsionalitas teknis penulisan teks, seperti: format, link, gambar, list, header dan lain-lain, dapat dilakukan dengan sangat mudah dan langsung pada editornya. Hal ini dimaksudkan agar anda bisa lebih fokus pada isi teks ketimbang penampilannya. Dokumentasi lengkap tentang Markdown dapat dibaca di website penciptanya, Mark Gruber, di: http://daringfireball.net/projects/markdown/syntax

Format Teks

Untuk menformat teks, kita dapat menggunakan kode:

  • *<teks>* untuk membuatnya italic,
  • **<teks>** untuk membuatnya bold,
  • ***<teks>*** untuk membuatnya bold dan italic,

Link

Untuk membuat link, anda cukup mengetikkan URL link tersebut dan Markdown akan secara otomatis merubahnya. Contoh, ketikkan: http://www.example.com dan Markdown akan merubahnya menjadi http://www.example.com

Untuk membuat teks yang mengandung link, seperti ini: contoh, anda dapat menggunakan kode: [teks](link).

Contoh: [teks dengan link](http://www.example.com) hasilnya: teks dengan link

Gambar

Untuk menyisipkan gambar, anda cukup mengetikkan kode: ![], Markdown akan menampilkan sebuah image placeholder di panel preview seperti ini:

Drag-and-drop file gambar dari komputer anda ke image placeholder yang disediakan. Gambar akan secara otomatis disisipkan.

Jika gambar yang mau disisipkan berada di Internet dan diacu dengan menggunakan URL, klik icon link pada bagian kiri bawah placeholder (lihat panah merah) lalu tuliskan URL gambarnya pada isian yang disediakan.

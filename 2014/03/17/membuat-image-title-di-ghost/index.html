Membuat image title di Ghost

Bisa dimaklumi jika banyak fitur standar belum ada dalam Ghost. Meskipun sudah cukup luas digunakan, Ghost masih dalam tahap awal perkembangannya. Penambahan fitur-fitur baru terjadi dengan cepat pada setiap updatenya.

Pada artikel ini, saya ingin mengulas satu fitur yang belum ada pada release Blog saat ini: Fitur image title. Seringkali kita ingin mencantumkan teks keterangan pada sebuah image atau mengidentifikasikan image kita dengan penomoran.

Beberapa cara dapat dilakukan untuk mengakali kekurangan ini, sambil menunggu Ghost.org menyediakan fitur image title secara resmi.

Pertama, kita dapat menggunakan tag HTML <small> setelah placeholder image, seperti ini:

<small>Ini contoh image title</small>

Ini hasilnya:
Ini contoh image title

Atau, karena Ghost mendukung pencantuman HTML code secara langsung pada post, kita dapat menggunakan tag <figure> dan <figcaption> pada HTML5. Berikut contohnya:

<figure>
    <img src='image.jpg' alt='missing' />
    <figcaption>Ini contoh image title</figcaption>
</figure>

Hasilnya seperti ini:

Saya sendiri lebih suka cara yang pertama karena tetap menggunakan image placeholder yang disedikan Ghost. Semoga fitur ini cepat ditambahkan pada update-update Ghost berikutnya. Sambil menunggu, saya memilih menggunakan tag <small> dan meminimalkan pemakaian image title hanya jika diperlukan saja.

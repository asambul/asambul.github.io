Markdown adalah modul editor di Ghost yang tugasnya menerjemahkan teks yang kita masukkan menjadi kode HTML. Dengan Markdown, berbagai fungsionalitas teknis penulisan teks, seperti: format, link, gambar, list, header dan lain-lain, dapat dilakukan dengan sangat mudah dan langsung pada editornya. Hal ini dimaksudkan agar anda bisa lebih fokus pada isi teks ketimbang penampilannya. Dokumentasi lengkap tentang Markdown dapat dibaca di website penciptanya, Mark Gruber, di: http://daringfireball.net/projects/markdown/syntax

Format Teks

Untuk menformat teks, kita dapat menggunakan kode:

  • *<teks>* untuk membuatnya italic,
  • **<teks>** untuk membuatnya bold,
  • ***<teks>*** untuk membuatnya bold dan italic,

Link

Untuk membuat link, anda cukup mengetikkan URL link tersebut dan Markdown akan secara otomatis merubahnya. Contoh, ketikkan: http://www.example.com dan Markdown akan merubahnya menjadi http://www.example.com

Untuk membuat teks yang mengandung link, seperti ini: contoh, anda dapat menggunakan kode: [teks](link).

Contoh: [teks dengan link](http://www.example.com) hasilnya: teks dengan link

Gambar

Untuk menyisipkan gambar, anda cukup mengetikkan kode: ![], Markdown akan menampilkan sebuah image placeholder di panel preview seperti ini:

Drag-and-drop file gambar dari komputer anda ke image placeholder yang disediakan. Gambar akan secara otomatis disisipkan.

Jika gambar yang mau disisipkan berada di Internet dan diacu dengan menggunakan URL, klik icon link pada bagian kiri bawah placeholder (lihat panah merah) lalu tuliskan URL gambarnya pada isian yang disediakan.

Forever adalah sebuah modul Node yang dapat digunakan untuk menjalankan Ghost secara permanen. Forever juga akan memonitor apakah Ghost berjalan atau tidak. Jika misalnya Ghost mengalami crash, Forever akan secara otomatis menjalankan Ghost kembali.

Instalasi Forever

Forever dapat di-instal dengan menjalankan command berikut:

 npm install -g forever

Jika gagal, kemungkinan karena anda butuh akses administrator untuk menjalankan perintah di atas. Coba lagi dengan menggunakan: sudo <command>

Menjalankan Ghost dengan Forever

Masuk dulu ke direktori Ghost, contoh: /Applications/XAMPP/htdocs/ghost pada Mac saya. Selanjutnya jalankan perintah berikut:

 forever start index.js

Anda dapat menutup command window, Ghost akan tetap berjalan di background.

Menghentikan Ghost dengan Forever

Pastikan anda berada pada direktori Ghost (lihat langkah sebelumnya), lalu jalankan perintah berikut:

 forever stop index.js

Setelah selesai menginstal Ghost di laptop saya, saya pikir perlu untuk membuat beberapa catatan. Biar nggak lupa, atau mungkin bisa berguna untuk kawan2 yang ingin mencoba bekerja dengan Ghost.

Berikut ini adalah langkah2 untuk bekerja dengan Ghost:

  1. Aktifkan HTTP server, dalam hal ini saya menggunakan XAMPP. Untuk bekerja dengan Ghost, anda cukup jalankan Apache servernya saja.
  2. Buka terminal window dan masuk ke directory Ghost, di laptop saya: /Applications/XAMPP/xamppfiles/htdocs/ghost
  3. Jalankan Ghost dengan cara: npm start di command line
  4. Login ke Ghost melalui browser: http://localhost:2368/ghost
  5. Buat postingan baru atau edit postingan lama. Jangan lupa meng-click tombol "Publish now"
  6. Pastikan anda berada pada directory Buster, contoh pada laptop saya: ~/buster/buster, lalu generate static pages dengan cara: buster generate --domain=http://127.0.0.1:2368.
  7. Upload ke Github dengan cara: buster deploy
  8. Tampilkan di browser dengan URL: http://asambul.github.io/ (lihat screenshot pada top image)

Catatan: Jika sebelumnya anda melakukan perubahan di remote repository sehingga isi direktori static di lokal berbeda dengan yang ada di remote (GitHub), masuklah ke direktori static dan jalankan perintah: git pull untuk men-sync repository lokal dan remote.

Sejauh ini, menurut saya Ghost cukup baik dan mudah digunakan. Kita hanya perlu membiasakan diri dulu dengan beberapa commands. Kalau sudah hafal, membuat postingan dapat dilakukan dengan cepat dan mudah.
